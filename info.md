# Локальный запуск

1. Установите зависимости:

```bash
npm install
```

2. Запустите фронтенд:

```bash
npm run dev
```

Сайт откроется на `http://localhost:5173`.

Опционально можно запустить API (если нужна подмена данных через `/api`):

```bash
npm run server
```

# Страницы и разделы

- `/` — главная страница с блоками: Hero, «О маршруте», интерактивная карта, все точки маршрута, партнеры и контакты.
- `/gallery/:id` — фотостраница выбранной точки маршрута.
- `/admin` — локальная админ‑панель для изменения ссылок на аудиодорожки (сохраняется в браузере).

Доступные `id` для галерей:
- `kanavinsky`
- `fair`
- `boulevard`
- `cathedral`
- `pavilion`

Дополнительно:
- `/#map` — прокрутка к интерактивной карте.
- `/?point=ID#map` — открывает карту и активирует конкретную точку.

Аудио по умолчанию берется из папки `audio/` и уже привязано к точкам маршрута.

# Карта (спутник Яндекс)

Для спутниковой карты можно добавить ключ Static Maps API:

1. Создайте файл `.env.local` в корне проекта.
2. Добавьте:

```bash
VITE_YMAPS_STATIC_KEY=ваш_ключ
```

Без ключа карта тоже работает, но у некоторых провайдеров может быть ограничение на загрузку спутниковых тайлов.
