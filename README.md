# Нижегородская Ярмарка — экскурсионный маршрут

## Локальный запуск

1. Установить зависимости:
   ```bash
   npm install
   ```

2. Запустить backend API в отдельном терминале:
   ```bash
   npm run server
   ```

3. Запустить фронтенд:
   ```bash
   npm run dev
   ```

4. Открыть в браузере:
   ```
   http://localhost:5173
   ```

## Примечания
- Фронтенд обращается к API по `/api/points` и `/api/partners` через прокси Vite.
- Аудио пока использует внешние демо-файлы; при необходимости замените `audio` на свои локальные файлы.
